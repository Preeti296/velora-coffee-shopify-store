## 📌 Project Overview
This is a fully responsive e-commerce website built using Shopify. The project focuses on creating a premium coffee brand experience with clean UI and smooth navigation.

## 🚀 Features
- Custom homepage design with hero, products, and about section
- Product listing with pricing and images
- Customer reviews integration (Judge.me)
- About and Contact pages
- Responsive and modern UI design

## 🛠 Tech Stack
- Shopify
- HTML, CSS, JavaScript (basic customization)
